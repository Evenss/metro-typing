# HANGZHOU METRO TYPING

杭州地铁站名打字练习。页面参考 Taiwan Metro Typing 的体验，以杭州 12 条运营线路、259 个车站和杭州行政区地图重做。

## 使用

1. 在首页选择线路。
2. 选择行驶区间、方向、中文或英文站名，以及「30 秒」或「全线」模式。
3. 点击「开始这条线路」，直接按画面上的站名逐字输入。
4. 输入正确时列车前进；输入错误会计入正确率并触发卡片抖动。

## 本地运行

```bash
npm install
npm run dev
```

打开 <http://localhost:3000>。

构建和测试：

```bash
npm test
```

线路与站名参考杭州地铁官方网站，数据快照更新至 2026-07-15。本项目不是杭州地铁官方服务，仅供打字练习。
