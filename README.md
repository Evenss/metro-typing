# METRO TYPING

简体中文 | [English](./README.en.md)

![METRO TYPING：今天，想坐哪一条线？](./public/og.png)

一个把城市地铁线路变成打字游戏的小项目。

你可以沿着真实的线路和站序输入站名，在游玩的过程中熟悉城市地铁线路。

## 已支持城市

- 杭州
- 上海
- 北京
- 深圳
- 成都

## 在线试玩

无需安装，使用 [**metro-typing.fun**](https://metro-typing.fun/) **在线试玩**。

喜欢这个项目的话，欢迎点一下右上角的 ⭐ Star！

## 玩法

1. 从地图或线路列表中选择一条地铁线路。
2. 选择行驶区间、方向，以及英文或拼音站名。
3. 选择游戏模式：
   - **30 秒**：在限定时间内尽可能多地完成站名。
   - **全线**：按照真实站序走完整条线路。
4. 点击「开始这条线路」，直接输入画面上显示的站名。英文和拼音中的空格均可输入或省略；拼音模式请使用英文键盘，声调无需输入，`ü` 输入为 `v`。
5. 每输入正确一个字符，列车就会继续前进；游戏会实时显示速度、正确率和完成进度。

## 本地启动

请先安装 Node.js 22.13 或更高版本，然后在项目目录中运行：

```bash
npm install
npm run dev
```

打开 <http://localhost:3000> 即可开始游玩。
