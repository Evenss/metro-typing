# METRO TYPING

简体中文 | [English](./README.en.md)

一个把城市地铁线路变成打字游戏的小项目。

你可以沿着真实的线路和站序输入站名，在游玩的过程中熟悉城市地铁线路。

## 已支持城市

- 杭州

## 在线试玩

无需安装，使用 [**metro-typing.fun**](https://metro-typing.fun/) **在线试玩**。

## 玩法

1. 从地图或线路列表中选择一条地铁线路。
2. 选择行驶区间、方向，以及中文或英文站名。
3. 选择游戏模式：
   - **30 秒**：在限定时间内尽可能多地完成站名。
   - **全线**：按照真实站序走完整条线路。
4. 点击「开始这条线路」，直接输入画面上显示的站名。
5. 每输入正确一个字符，列车就会继续前进；游戏会实时显示速度、正确率和完成进度。

## 灵感来源

本项目的创意、玩法和界面设计参考了 [TAIWAN METRO TYPING｜台灣站名打字練習](https://tw-metro-typing.yencheng.dev/)，并在此基础上尝试支持更多城市的地铁线路。感谢原作者带来的优秀作品与灵感。

## 本地启动

请先安装 Node.js 22.13 或更高版本，然后在项目目录中运行：

```bash
npm install
npm run dev
```

打开 <http://localhost:3000> 即可开始游玩。
